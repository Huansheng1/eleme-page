# vue-eleme

本项目采用了`Vue`全家桶制作的单页面SPA应用

**所用技术栈**：

* `Vue`全家桶
> `Vue`、`VueX`、`VueRouter`、`VueCli`
* `axios`网络请求模块
> 网络请求接口的封装
* `mockjs`模拟数据请求
> 通过拦截网络请求返回项目内置的JSON数据
* `cube-ui`UI库
> 使用`createAPI`创建调用支付订单弹窗
* `better-scroll`滚动库
> 页面商品局部滚动与跳转
* ...
